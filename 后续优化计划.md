## 一、快速优化（低投入高回报）

### 1.1 真实 Google Calendar API 接入

**现状**：使用本地 SQLite 模拟日历，数据是种子数据。

**目标**：接入 Google Calendar API v3，操作真实日历。

**做法**：
```python
# services/google_calendar.py — 新文件，实现与 LocalCalendarService 相同的接口
class GoogleCalendarService:
    def list_events(self, start, end) -> List[dict]: ...
    def get_event(self, event_id) -> dict: ...
    def update_event(self, event_id, updates) -> bool: ...
    def create_event(self, event) -> str: ...
    def delete_event(self, event_id) -> bool: ...
    def check_conflicts(self, start, end, exclude_id=None) -> List[dict]: ...
```

**步骤**：
1. Google Cloud Console → 创建项目 → 启用 Calendar API → 下载 `credentials.json`
2. 实现 OAuth 2.0 认证流程
3. 实现 `GoogleCalendarService`（可参考 `LocalCalendarService` 的接口）
4. 在 `config/settings.py` 加开关：`CALENDAR_BACKEND=local|google`
5. 修改 `task_planning.py` / `execution_engine.py` 按开关选择后端

**预计时间**：1-2 天

### 1.2 Web 界面（FastAPI + 简单前端）

**现状**：只有 CLI，不够直观。

**目标**：Web 聊天界面，浏览器里输入指令、查看结果。

**技术选型**：
- 后端：FastAPI（轻量、异步、自带 Swagger）
- 前端：单个 HTML 文件 + htmx（无需 React/Vue）
- 或者：Streamlit（最快出原型，但不适合生产）

**最小实现**：
```python
# web_server.py
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class CommandRequest(BaseModel):
    command: str

@app.post("/api/chat")
def chat(req: CommandRequest):
    response = run_agent(req.command)  # 复用 CLI 的 run_agent
    return {"response": response}
```

**步骤**：
1. `pip install fastapi uvicorn`
2. 创建 `web_server.py`，暴露 POST `/api/chat`
3. 创建 `static/index.html`，简单的聊天 UI
4. `python web_server.py` → 浏览器访问 `http://localhost:8000`

**预计时间**：1 天（FastAPI）或 3 小时（Streamlit）

### 1.3 会议链接自动生成

**现状**：创建的会议没有线上链接。

**目标**：创建事件时自动附带线上会议链接。

**支持的平台**：
- 腾讯会议 API
- Zoom API
- Google Meet（如果用 Google Calendar，自带）

**最简单方案**：
```python
# 在 execution_engine.py 的 create_event 分支中：
def generate_meet_link(title: str) -> str:
    """最简单的方案：生成一个 Jitsi Meet 链接（免费、无需API）"""
    import hashlib
    room_id = hashlib.md5(title.encode()).hexdigest()[:12]
    return f"https://meet.jit.si/{room_id}"
```

**预计时间**：半天

```python
# 使用 Web Speech API，浏览器原生支持中文语音识别

# pip install SpeechRecognition pyaudio
import speech_recognition as sr

def listen_and_execute():
    r = sr.Recognizer()
    with sr.Microphone() as source:
        audio = r.listen(source)
    text = r.recognize_google(audio, language='zh-CN')
    return run_agent(text)
```

**推荐**：方案A（Web界面 + 浏览器语音），零额外依赖。
